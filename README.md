# CS155-Repeating-Calculator
This is a calculator that constantly asks for inputs based on user input, and then calculates division, addition, subtraction, mulitplication, and even raising to a power.

## Usage
Just a calculator that handles positive integers and can display up to three digit answers

## Requirements
No specific requirements needed, just the LC3 Assembly simulator

## Installation
Download either the .asm or the .obj file depending, .asm is for assembly and .obj is for binary. After downloading open the simulator, load the file, save and assemble. Set the breakpoint on HALT found in x3078 and you're good to go.
